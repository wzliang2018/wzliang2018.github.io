Hexo文件迁移

发表于 2018-08-12 | 分类于 Hexo使用

这几天在重装电脑系统，将用不惯的Win10换成了习惯的Win7，所以Hexo就得重新捣鼓。
整个过程挺简单的，基本步骤如下：

1.将themes, source, _config.yml 这3个文件拷贝出来放好；

2.按照可能是最详细的 Hexo + GitHub Pages 搭建博客的教程 这篇文档重新搭建Node.js和Git的环境，Github配置，安装配置Hexo，关联 Hexo 与 GitHub Pages就OK了;
3.把前面拷贝好的文件拷贝到文件路径进行替换。

如果是同一台电脑更换不同的文件路径，那就更简单了，在将要放置博客文件的路径下点击右键，选择Git Bash Here，然后将原来路径下博客的全部文件拷贝到该路径下，完成。
